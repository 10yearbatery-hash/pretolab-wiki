팀 wiki 주간 중복·충돌 정리 (기덕 전용, 주 1회)

다음 순서로 실행:

1. .omc/wiki/ 전체 파일 목록과 제목 파악
2. 내용이 겹치거나 중복되는 페이지 찾기
3. 발견된 중복 목록을 보여주고 처리 방법 제안:
   - 합치기: 두 페이지 내용을 하나로 통합 후 나머지 삭제
   - 업데이트: 오래된 버전 삭제, 최신 버전 유지
   - 유지: 내용이 달라서 둘 다 필요한 경우
4. 확인 후 처리 실행
5. 완료 후 git add .omc/wiki/ && git commit -m "wiki: 주간 정리" && git push
