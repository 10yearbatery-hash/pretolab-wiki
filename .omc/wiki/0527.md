# 현장조사 보고 - 공인중개사 시장 탈락 (영환, 0527)

## 개요
- **작성자**: 영환
- **날짜**: 2026-05-27
- **방문지**: 태원중개법인 강남 (직원 약 10명 규모)
- **결론**: **공인중개사 시장 탈락**

## 현장 워크플로 파악

공인중개사의 실제 업무 흐름:
1. **매물 수집**: 공실클럽 앱으로 매물 파악
2. **등기 확인**: 세움터 / 부동산써브로 등기부등본 확인
3. **광고 등록**: 네이버부동산 / 직방 / 다방 등 플랫폼에 매물 등록

## 탈락 이유 3가지

### 1. 레드오션
- 이미 크리에이지AI 등 선행 솔루션이 시장을 점유
- 차별화 진입 포인트 없음

### 2. user ≠ buyer 구조
- 실제 도구를 사용하는 사람(직원)과 결제 의사결정자(중개법인 대표)가 다름
- 직원 설득 → 대표 설득 → 계약의 이중 허들 존재

### 3. ROI 불명확
- 도구 도입 시 얼마나 시간/비용이 절감되는지 수치화 어려움
- 중개업 특성상 "매물 성사"가 ROI이므로 AI 도입 효과 측정 어려움

## 다음 시장 선정 기준 (탈락 후 도출)

| 기준 | 설명 |
|------|------|
| user = buyer | 도구 사용자가 곧 결제 의사결정자 |
| ROI 명확 | "이 도구 써서 이만큼 절감/수익" 계산 가능 |
| 도입 거부감 없음 | 연령/직군이 디지털 도구에 친숙 |

## 다음 단계 (결정)
