# 최종 아이템 확정 - 사진 미러링 앱

**날짜**: 2026년 6월 1일 | **참석자**: 기덕, 영환, 건준

## 결정 경위

### 기덕 아이디어 → 발전 → 탈락
- 기덕의 "자동 브이로그 편집 앱"을 구체화하는 과정에서
- **"전문가가 만든 템플릿에 영상만 끼워넣으면 자동 편집"** 방식으로 발전
- 예: 여행 쇼츠, 브이로그 등에 적용
- 그런데 **CapCut이 동일한 기능을 이미 제공 중** → 차별화 불가 → 탈락

### 영환 아이디어로 최종 결정
- 위 탐색 과정을 거치며 팀 전체가 **영환의 사진 미러링 앱**을 최종 선택

## 최종 아이템: 사진 미러링 앱

### 핵심 컨셉
사진 잘 찍는 사람(디렉터)이 찍히는 사람의 폰으로 카메라 화면을 실시간 미러링하여 디렉팅하는 서비스.

| 역할 | 경험 |
|------|------|
| 디렉터 (잘 찍는 사람) | 구도·빛·포즈를 실시간으로 보며 디렉팅 |
| 찍히는 사람 | 디렉터 화면을 실시간으로 보며 포즈 조정 |

### 기존 컨셉과의 차이
- 기존([[2]]): "찍히는 사람이 포즈 맞추기 위해 화면을 보는 것"에 포커스
- **확정 컨셉**: "잘 찍는 사람이 디렉팅하는 경험" 자체에 포커스
