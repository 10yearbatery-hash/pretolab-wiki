# 6월 1일 — 아이디어 발표 회의

B2C 피벗 후 각자 아이디어를 조사해서 발표한 회의.

## 기덕 아이디어 (4개)

### 아이디어 1: 아이돌 통합 스케줄 & 모먼트 피드
- 공식 스케줄과 팬들이 좋아할 순간들을 그룹별로 모아 한눈에 보여주는 서비스
- **장점**: K-POP 글로벌 수요 증가, 덕후에게 실제로 필요한 서비스
- **단점**: 공식 팬클럽 채널로도 충분히 커버 가능하다는 반론

### 아이디어 2: 자동 브이로그 편집 앱
- 앱 내에서 영상을 찍으면 긴 영상도 자동으로 브이로그 형식으로 편집
- **장점**: 20대 여성/브이로그 희망자라는 구체적 타겟, 일상 공유 니즈 검증된 시장
- **단점**: 자동 편집 기술 구현 가능성 불확실

### 아이디어 3: AI 대화형 학습 도구
- 자료를 올리면 AI가 대화로 가르쳐주는 학습 서비스
- **장점**: 대학생 시험기간이라는 명확한 사용 시점, 단톡방 공유로 바이럴 구조 형성
- **단점**: ChatGPT 프롬프팅으로 대체 가능 → 차별점 약함, 장기 생존 가능성 낮음

### 아이디어 4: 스터디 기반 자기계발 지속 서비스
- 보증금, 벌금, 패스 제도를 활용한 자기계발 스터디
- **장점**: 직접 많이 발전시켜온 아이디어, MVP 데이터 보유 중
- **단점**: 기술 구현 난이도 및 시장 매력도 고민 필요

# 최종 아이템 결정 - 사진 미러링 앱 (0601)

## 최종 선택 아이디어: 사진 미러링 앱

### 핵심 컨셉
사진을 잘 찍는 사람(디렉터)이 찍히는 사람의 폰으로 카메라 화면을 실시간 미러링하여 디렉팅하는 서비스.

- **디렉터**: 구도·빛·포즈를 실시간으로 보며 "조금 왼쪽으로" 같은 디렉팅 가능
- **찍히는 사람**: 디렉터의 화면을 실시간으로 보며 포즈 조정 가능

### 기존 아이디어와의 차이
기존 wiki([[2]])의 컨셉은 "찍히는 사람이 포즈 맞추기 위해 보는 것"에 포커스.  
최종 확정 컨셉은 **디렉터가 디렉팅하는 경험** 자체에 포커스가 이동.
