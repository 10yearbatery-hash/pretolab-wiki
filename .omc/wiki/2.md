# 아이디어 분석 - 사진 미러링 앱 (영환, 최종 후보 2위)

> 작성자: 영환 | 현재 상태: ✅ 최종 후보 2위 | 출처: photo-mirroring.pdf

## 개념

사진 찍히는 사람의 폰으로 찍는 사람 화면을 실시간 미러링. 포즈 맞추기 편하게.

## 레퍼런스

- **ph02 max** — 리뷰 191개, 평점 4.75점
- 전 세계에 동일 기능 소프트웨어 없음 → **소프트웨어 공백 최강**

## 강점 평가표

| 항목 | 평가 |
|------|------|
| 소프트웨어 공백 | ✅ 전 세계 없음 |
| 경쟁 위험 | ✅ 낮음 |
| WTP | ❓ 앱에 낼지 미검증 |
| 리텐션 | ❌ 여행/외출 때만 사용 |
| 개발 난이도 | 중간 (WebRTC) |

## 타겟별 방향

| 타겟 | 제품 방향 | 수익 |
|------|----------|------|
| 인스타 진심 B2C | QR 링크, 공유 카드 바이럴 | 무료 → 구독 |
| 보세쇼핑몰 B2B | 동대문 직접 영업, 세션 관리 | 월 구독 2-3만원 |
| 협찬 인플루언서 | 반전문가, 커뮤니티 | 중간 |

## 리스크

- WTP 미검증 — 앱에 돈 낼지 불확실
- 사용 빈도 낮아 데모데이까지 실적 증명 어려움
- 기술: WebRTC 오픈소스 기반, 기술 해자 없음

## Wizard of Oz 테스트 계획

- **대상:** 인스타 진심인 여사친 2-3명
- **방법:** Zoom 화면공유로 찍는 사람 화면 → 찍히는 사람 폰으로 보기
- **주의:** 테스트 전 "앱 만들려고"라고 말하지 말 것 (편향 방지)
- **성공 기준:** 3명 중 2명이 포즈 실제로 수정하거나 "이거 편하다" 반응
- **필수 질문:** "앱 있으면 얼마 낼 것 같아?" (WTP 직접 확인)
