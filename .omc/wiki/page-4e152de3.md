# 현장 방문 인터뷰 준비 가이드 - 공인중개사편 (영환)

## 개요
- **작성자**: 영환
- **용도**: 강남 중개법인 현장 방문 사전 준비 가이드
- **실제 방문 결과**: [[0527]] 참조

## 3단계 현장 방문 프로세스

### Phase 1. 오전 - 관찰
관찰 포인트:
- 등기부등본 발급 과정
- 광고 등록 (네이버/직방/다방) 방식
- 매물 정보 관리 방법
- 고객 응대 방식

목표: 말보다 행동으로 실제 업무 흐름 파악

### Phase 2. 점심 - Debrief
- 오전 관찰 내용 정리
- 핵심 질문 리스트 정제
- "왜 저렇게 할까?"에 대한 가설 세우기

### Phase 3. 오후 - 인터뷰
직원 또는 대표와 직접 대화

## 좋은 데이터 vs 나쁜 데이터

| 좋은 데이터 ✅ | 나쁜 데이터 ❌ |
|--------------|--------------|
| 구체적 숫자 ("하루에 몇 건") | 가정 ("아마도 ~") |
| 과거 행동 ("지난달에 이렇게 했어요") | 칭찬 ("좋을 것 같아요") |
| 감정 담긴 표현 ("진짜 귀찮아요") | 일반론 ("다들 그럴 거예요") |
