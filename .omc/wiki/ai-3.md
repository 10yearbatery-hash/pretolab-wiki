# 아이디어 분석 - AI 헬스 체형 시각화 (영환, 3위 후보)

> 작성자: 영환 | 현재 상태: 🔍 3위 후보 (미검증) | 출처: ai-body-visualization.pdf

## 개념

거울 셀카 찍어 등록 → "이 루틴 3개월 하면 몸이 이렇게 변합니다" AI 합성 이미지 생성. 매주 실제 사진과 AI 예측 비교로 리텐션 유지.

## 레퍼런스

- 직접 발굴 아이디어 (해외 직접 레퍼런스 없음)
- 유사 사례: FaceApp 바디 변환 기능

## 핵심 인사이트

헬스 앱의 이탈 구간은 **3-4주차** — 아직 몸의 변화가 눈에 보이지 않는 시점.  
AI 시각화로 이 구간을 브리징하면 리텐션 해결 가능.

## 강점 평가표

| 항목 | 평가 |
|------|------|
| 바이럴 잠재력 | ✅ Before/After AI 이미지 SNS 공유 |
| 리텐션 훅 | ✅ 매주 예측 vs 실제 비교 |
| 시장 공백 | ❓ 앱스토어 경쟁자 미확인 |
| WTP | ❓ 미검증 |
| 기술 난이도 | 중간 |

## 바이럴 훅

- Before/After AI 이미지 인스타/틱톡 공유
- "3개월 후 내 몸 AI 예측" 포맷 → 반복 가능

## 리스크

- AI 합성 퀄리티 낮으면 역효과
- 경쟁자 미확인 (앱스토어 검색 필요)
- 기술 난이도 중간 — 기덕 feasibility 확인 필요
