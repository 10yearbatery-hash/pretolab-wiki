# 0526 회의록 - 시장별 인터뷰 전략 수립

## 기본 정보
- **날짜**: 2026-05-26
- **시간**: 15:01 ~ 15:53
- **참석자**: 기덕, 건준(형), 영환
- **목적**: 각자 조사할 시장(타겟) 공유 및 유저 인터뷰 전략 논의

## 1. 기덕 — 1인 크리에이터 / 유튜브 영상 시장

### 선정 이유
- 콘텐츠 시장이 1인 셀러 시장 대비 10배 이상 크다고 판단
- 유튜브 산업은 앞으로도 계속 성장할 것
- 본인이 가장 잘 아는 영역

### 인터뷰 방향
- "유튜브를 하고 싶지만 못하는 사람" + "이미 하고 있지만 어려움을 겪는 사람" 두 트랙
- 기획 / 편집 등 어디서 병목이 생기는지 구체적 과정 관찰
- 문제 해결을 위해 돈을 써본 적이 있는지 확인

### 피드백 / 논의
- 영역은 괜찮지만 타겟이 너무 넓음 → **세분화 필요**
- 영상 편집 시장은 캡컷·브루 등 대기업 포화 상태 → **기획 쪽이 더 유망할 수 있음**
- 개인 유튜버는 B2C라 지갑 열기 어려움 → 기업 유튜브팀·마케터 같은 B2B 쪽도 함께 고려

### 결정
- 타겟을 세분화해서 각 카테고리당 1명씩 인터뷰
- B2B 전문가(PD, 편집자 등) 콜드메일도 시도

## 2. 건준 — 헬스 트레이너 (외국인/단체 → 전환)

### 초기 안: 외국인 / 동아리 단체 운영
- 접근 쉬운 시장 기준으로 선정
- 인터뷰 전략은 매우 디테일하게 짜둠 (스몰토크 → 공감대 → 병목 발굴)

### 전환 사유
- 외국인은 몇 개월 체류 후 떠나는 구조 → **LTV 우려**
- 동아리 운영은 시장 자체가 작고 비즈니스 지속성 약함
- 헬스 트레이너는 카톡으로도 빠르게 접근 가능, 자영업 영역(미용사·필라테스 등)으로 확장성도 큼

### 결정
- **헬스 트레이너로 시장 변경**
- 기존 인터뷰 플로우(스몰토크 → 공감 기반 병목 탐색 → 오픈 발굴)는 그대로 가져가되 타겟에 맞게 수정

## 3. 영환 — 공인중개사 시장

### 계획
- 다음날 공인중개사 사무실 방문
- 오전: 업무 흐름 관찰 + 중간중간 질문
- 점심: 1차 인터뷰 (관찰 기반)
- 오후: 동료 직원 1인당 10분 짧은 인터뷰
- 중개법인 대표 인터뷰도 시도

### 한계 / 가능성
- 전국 사무소 약 10만 개로 시장이 작은 편
- 그러나 여기서 발견된 문제가 사무 업무 전반으로 확장 가능

## 주요 공통 논의

> **"아이디어보다 문제 발굴이 우선"** — 현재 단계는 아이디어 도출이 아닌 시장/유저 이해

- **시장 크기에 너무 매몰되지 말 것**: 구글·에이블리 사례처럼 경쟁자 있어도 충분히 진입 가능
- **인터뷰는 The Mom Test 원칙대로**: 자연스러운 대화 흐름, 공감 기반 질문, 오픈 발굴
- **"이 시장은 안 된다"는 결론도 성과**: $50K 쓸 걸 $5K로 검증하는 것이 시장 조사의 목표
