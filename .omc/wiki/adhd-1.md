## 개념

ADHD 뇌에 맞는 시각적 플래너. 색깔 블록 타이머, AI 태스크 분해, 루틴 관리.

## 레퍼런스

- **Tiimo** (덴마크) — Apple 2025 올해의 앱, 유료구독 5만명, 펀딩 $4.8M
- **WTP: $54/년** ✅ 확인됨

## 왜 지금인가

- 한국 ADHD 진단 4년새 80% 증가, 성인은 10배 증가
- 영어로 Tiimo 쓰는 한국인 이미 존재 → 한국어 특화 공백 확인됨
- 일반 할 일 앱은 ADHD 유저에게 "맞지 않음" → Must-Have 수준

## 강점 평가표

| 항목 | 평가 |
|------|------|
| WTP | ✅ $54/년 확인됨 |
| 리텐션 | ✅ 매일 사용 |
| 개발 난이도 | ✅ 낮음 (UI 중심) |
| 데모데이 실적 | ✅ DAU 데이터 바로 쌓임 |
| 소프트웨어 공백 | 중간 (Tiimo 있음, 한국어만 약함) |

## 리스크

- Tiimo가 한국어 추가할 수 있음
- 팀이 ADHD를 직접 이해하는가
- 경쟁 위험: 중간
