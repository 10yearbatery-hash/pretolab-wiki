# 헬스트레이너 인터뷰 방법론 가이드 (건준)

## 개요
- **작성자**: 건준 (권준영)
- **대상**: 헬스트레이너 친구 / 지인 대상 대화 가이드
- **목적**: 실제 페인포인트를 캐내는 구조화된 대화 흐름

## 6단계 대화 흐름

### Step 0. 근황 토크
- "요즘 어때?" "헬스장 분위기 어때?"
- 목적: 경계 허물기, 자연스러운 대화 시작

### Step 1. 병목 훑기
- "트레이닝 하면서 제일 귀찮은 게 뭐야?"
- "하루 중 가장 에너지 낭비라고 느끼는 순간?"
- 목적: 업무 흐름의 마찰 포인트 파악

### Step 2. Normalize
- "다른 트레이너들도 그런 거야, 아니면 너만 그래?"
- 목적: "이게 원래 이런 거야"라는 체념을 걷어내기

### Step 3. 사건 캐기 (핵심)
- "최근에 그 상황이 진짜 짜증났던 순간 있어? 언제?"
- "그때 어떻게 했어?"
- 목적: 추상적 불만 → 구체적 사건으로 전환

### Step 4. 능력 증명
- "그거 어떻게 해결하려고 해봤어?"
- "나한테 고맙다고 할 수준으로 해결해주면 뭐가 필요해?"
- 목적: 해결 의지와 현재 대안 파악

### Step 5. 오픈 발굴
- "내가 안 물어봤는데 사실 제일 힘든 게 따로 있어?"
- 목적: 예상 못 한 Pain Point 발굴

## 절대 금지 사항

| 금지 행동 | 이유 |
|----------|------|
| 미래/가정형 질문 ("이런 기능 있으면 쓸 것 같아?") | 실제 행동 데이터가 아닌 희망 사항만 얻게 됨 |
| 공감으로 끝내기 ("아 그렇구나 힘들겠다") | 문제가 얼마나 심각한지 정량화 안 됨 |
| 칭찬을 데이터로 착각 ("좋은 것 같아요!") | 예의상 하는 말과 진짜 Pain Point를 구분 못 함 |

## 좋은 데이터 vs 나쁜 데이터

| 좋은 데이터 ✅ | 나쁜 데이터 ❌ |
|--------------|--------------|
| 구체적 숫자 ("하루에 3번씩") | 가정 ("아마도 ~할 것 같아") |
| 과거 행동 ("지난주에 이렇게 했어") | 칭찬 ("좋을 것 같은데") |
| 감정 담긴 표현 ("진짜 짜증 났어") | 일반론 ("다들 그럴 거야") |
